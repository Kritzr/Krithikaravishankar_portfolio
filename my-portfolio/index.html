Networks Project

Developed a Wireless Body Area Network (WBAN) simulation to study secure data routing in healthcare communication systems. Implemented algorithms to compute the shortest communication path between wearable sensor nodes while detecting and mitigating blackhole attacks, which attempt to drop or misroute packets. This project highlighted the importance of trust-aware routing protocols in safeguarding patient monitoring networks
